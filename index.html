Web Framework

This page is to assist in the production and branding of Bates Technical College websites.

  • Bootstrap
  • Colors
  • Buttons
  • Hover Module

Currently, the framework of choice is Twitter Bootstrap.

Bootstrap supports millions of websites across the web. The Bootstrap framework provides fast scaffolding for designs and a tested grid layout for mobile first design. Bates uses Bootstrap to ensure that our websites are meeting modern accessibility standards and reaching as many students and partners as possible.

To download the latest version of Bootstrap and read the Bootstrap documentation, visit the Bootstrap website.

These are the Bates Technical College Buttons

To style buttons properly, inspect element to find the correct classes to use.

Below is an example of the button html. 

 <button class="button bates-button bates-dark-blue-button">...</button>

To create hollow buttons, add a .hollow class to the button

 <button class="button hollow bates-button bates-dark-blue-button">...</button>

To create rounded buttons, add a .bates-rounded class to the button

 <button class="bates-rounded button bates-button bates-dark-blue-button">...</button>

Sometimes a standard link will not do. We can use an Image Module

<div class="img-hover-mod clearfix">
  <div class="row">
    <div class="col-xs-12 col-sm-6 col-md-4"> <!-- Change columns based on design need -->
      <figure class="figure fig-hover bates-dark-blue-background">
        <div class="figure-img"style="background-image:url('...')" alt="..."></div>
        <figcaption class="figure-caption">
          <h4>...</h4>
          <button class="button bates-button bates-green-button">
            ...
          </button>
        </figcaption>
      </figure>
    </div> <!-- Ends bootstrap columns -->
  </div> <!-- Ends row -->
</div> <!-- Ends img hover mod -->

To change the height of the modules, change the pixel amount in the style.scss file. 

.img-hover-mod{
  @include bates-hover-mod(225px);
}
